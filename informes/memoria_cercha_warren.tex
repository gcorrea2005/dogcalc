%============================================
\section{CONCLUSIONES}
%============================================

\begin{enumerate}
    \item La cercha Warren en caj\'on 3D es estructuralmente estable.
          El an\'alisis FEM confirma equilibrio para las 12 combinaciones.

    \item El perfil \textbf{TUBO 70$\times$70$\times$4 mm es INSUFICIENTE}
          para el combo 105 (1.2D + 1.6G + 0.5W). El cord\'on superior
          alcanza 459.4 kN contra 224 kN de capacidad (ratio = 2.05).

    \item \textbf{Se recomienda} aumentar secci\'on a:
          \textbf{TUBO 100$\times$100$\times$5 mm} ($A = 1,900$ mm\textsuperscript{2},
          $\phi_c P_n \approx 400$ kN).

    \item El desplazamiento m\'aximo ($\approx 55$ mm, $L/300$) es aceptable
          para cubierta sin requisitos estrictos de servicio.

    \item Las conexiones se modelaron r\'igidas. Para an\'alisis m\'as
          conservador, considerar liberaciones de momento en diagonales.

    \item Verificar uniones soldadas y placa base seg\'un NSR-10 F.2.
\end{enumerate}
